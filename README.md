# 🍽️ Recipe Sharing Platform
### A Full-Stack Web Application for Sharing & Discovering Recipes

*A community-driven recipe platform where users can register, share their own recipes, browse others', search by interest, and leave ratings — built from scratch with a vanilla web stack.*

## 🔍 Overview

Recipe Sharing Platform is a **full-stack web application** built as a group project, delivering a complete community space for food lovers to share, discover, and interact with recipes. The platform is built entirely with a vanilla web stack — no frameworks — demonstrating a solid command of core web development fundamentals from database design through to interactive frontend.

Users can create accounts, publish their own recipes, manage their content, search and browse the community's collection, and engage through ratings and comments.

---

## ✅ Features

| Feature | Description |
|---|---|
| 🔐 **User Registration & Login** | Secure account creation and session-based authentication |
| 📝 **Create / Edit / Delete Recipes** | Full CRUD operations — users own and manage their content |
| 🔍 **Browse & Search Recipes** | Explore the full recipe collection and search by keyword |
| ⭐ **Comments & Ratings** | Community engagement through reviews and star ratings |

---

## 🏗️ Tech Stack

| Layer | Technology |
|---|---|
| **Frontend** | HTML, CSS, JavaScript (vanilla) |
| **Backend** | PHP |
| **Database** | MySQL |

No external frameworks — built from scratch to demonstrate full understanding of how the web stack fits together end to end.

---

## 🗂️ How It Works

```
Browser (HTML / CSS / JS)
        │
        │  HTTP Requests
        ▼
   PHP Backend
   ├── Authentication  — register, login, session management
   ├── Recipe CRUD     — create, read, update, delete recipes
   ├── Search          — query recipes by keyword
   └── Ratings & Comments — store and display user interactions
        │
        │  SQL Queries
        ▼
   MySQL Database
   ├── users
   ├── recipes
   ├── comments
   └── ratings
```

## 🚀 Getting Started

### Prerequisites
- A local server environment: **XAMPP**, **WAMP**, or **MAMP**
- PHP 7.x or higher
- MySQL

### Setup

```bash
# 1. Clone the repository
git clone https://github.com/omardiab9951/Recipe-Sharing-Platform-.git

# 2. Move the project to your server's root directory
#    e.g. for XAMPP: C:/xampp/htdocs/Recipe-Sharing-Platform

# 3. Import the database
#    Open phpMyAdmin → create a new database → import the provided .sql file

# 4. Configure the database connection
#    Edit the DB config file with your local credentials (host, user, password, db name)

# 5. Start Apache & MySQL in XAMPP/WAMP, then visit:
#    http://localhost/Recipe-Sharing-Platform
```

## 🎯 Key Learning Outcomes

This project puts the following web development fundamentals into practice:

- **Server-side scripting** with PHP — handling requests, sessions, and business logic
- **Relational database design** — structured MySQL schema with multiple related tables
- **CRUD operations** — full create, read, update, delete cycle for user-generated content
- **User authentication** — registration, login, and session management without external libraries
- **Dynamic HTML rendering** — PHP-generated pages served based on database state
- **Frontend interactivity** — vanilla JavaScript for client-side behavior
